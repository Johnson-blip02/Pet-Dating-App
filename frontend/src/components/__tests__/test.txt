| Component Type      | What to Test                                           |
| ------------------- | ------------------------------------------------------ |
| **UI Components**   | Renders correct content, reacts to props, classNames   |

| **Router Pages**    | Navigation behavior, conditional rendering             |
| **Chat/Socket UI**  | Message rendering, button actions, disconnect warnings |
| **Header/Footer**   | Logged-in state display, theme toggle, link navigation |
